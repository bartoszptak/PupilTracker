## Requirements
* webcam
* python >= 3.6.4
* keras >= 2.1.6
* tensorflow/tensorflow-gpu >=1.9
* numpy >= 1.11.3
* opencv-python >= 3.4.1
* dlib >= 19.15.0
* imutils >= 0.4.6
